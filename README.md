# ms-model-converter

Tools, configs and documentation for agents to migrate PyTorch models to MindSpore and to develop models in MindSpore. The repository focuses on conversion utilities, API mappings, tests and examples for popular Hugging Face projects (for example, `transformers`).

Exploration
* mini-swe, with cluade-code-sonnet-4.5 model
* Claude Code pack

See the `mini-swe/` folder for the legacy mini-swe agent knowledgebase, conversion scripts and usage guides.

ms-model-converter/
├── mini-swe/           # mini-swe agent knowledgebase and configs
├── claude-code/        # self-contained claude-code packs for conversion
└── README.md
